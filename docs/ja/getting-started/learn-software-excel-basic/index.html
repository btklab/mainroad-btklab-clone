はじめに

「Microsoft Excel」の基礎知識。関数、テーブル、ピボットテーブルで仕事をパズル化しよう。

前提条件

  • ✓ 「Microsoft Excel 」のインストール、またはクラウドストレージ「Microsoft OneDrive」経由でのWeb版Excelの利用
  • ✓ 検証した環境はこちら

用語

Microsoft Excel

Microsoft Excelは日本の会社で広く使われている表計算ソフトウェア。仕事をExcelに落とし込むことができれば、仕事中にもパズルを楽しめる。

スプレッドシートは日本語で「表計算」。「表」はTableを意味し、「計算」はCalculationを意味する。
